# erii manage

数据管理工具，用于查看和编辑 erii 运行时产生的数据。

```bash
erii manage
```

## 可管理的数据

| 类型       | 说明                                                   |
|:---------|:-----------------------------------------------------|
| **事实记忆** | erii 记住的事实（群友生日、群约定等），进入后包含 Record、Vector、Graph 三个视图 |
| **用户画像** | 群友的兴趣爱好、说话风格                                         |
| **表情包**  | 群内流行语、常用表情                                           |
| **词汇**   | erii 学到的新词、黑话                                        |
| **摘要**   | 每日聊天总结、话题回顾                                          |
| **聊天历史** | 原始聊天记录                                               |
| **资源**   | 群内分享的图片和文件                                           |
| **情绪状态** | erii 当前的情绪数据                                         |
| **心流状态** | erii 当前的话题投入度                                        |
| **定时任务** | 已设置的提醒和任务                                            |

进入管理界面后，选择 Bot 和群组即可对该群的数据进行查看、编辑和删除。

## Memory 子菜单

| 视图         | 说明                                    |
|:-----------|:--------------------------------------|
| **Record** | 原有 Memory 表格列表，可查看、搜索、预览、创建、编辑和删除事实记忆 |
| **Vector** | 左侧输入文本，右侧展示嵌入搜索命中的事实和分数               |
| **Graph**  | 左侧输入文本，右侧展示嵌入召回结果，并显示一跳实体图的节点和边       |
