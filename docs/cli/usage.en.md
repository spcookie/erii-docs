# `erii usage`

Token usage statistics and cost analysis TUI panel.

```bash
erii usage
```

## Charts

- **Today / Accumulated Totals** — Input Hit, Input Miss, Output, Cost, and Hit Rate KPIs
- **Scene Tokens** — Token consumption breakdown by scene (Chat, Search, Plugin, Memory, etc.)
- **Model Tokens** — Token consumption breakdown by LLM model
- **Weekly Trend** — Daily token usage trend line chart (last 7 days)
- **Daily Intensity** — Calendar heatmap showing daily token usage intensity over 6 months

## Scope

Use `tab` to switch between different scopes:

- All Bots — aggregated usage across all bots
- Single Bot — usage for a specific bot
- All Groups — aggregated usage across all groups for a bot
- Single Group — usage for a specific group of a bot
