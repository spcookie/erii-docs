# `erii setup`

Interactive setup wizard that guides users through initial configuration.

```bash
erii setup
```

## Setup Flow

1. **LLM Configuration** -- Select an LLM provider (OpenAI-compatible / Anthropic-compatible), and fill in the API Key
   and Base URL
2. **Embedding Configuration** -- Select a vector embedding service and fill in the API Key
3. **Search/Vision Configuration** -- (Optional) Configure search and vision analysis services
4. **OneBot Connection** -- Fill in the NapCat WebSocket address and Token
5. **Bot Role** -- Select or create a Bot role
6. **Group Configuration** -- Configure enabled groups
