# `erii usage`

Token 用量统计与成本分析 TUI 面板。

```bash
erii usage
```

## 图表

- **今日 / 累计汇总** — Input Hit、Input Miss、Output、Cost、Hit Rate 等 KPI 指标
- **Scene Tokens** — 按场景（聊天、搜索、插件、记忆等）的 Token 消耗分布
- **Model Tokens** — 按 LLM 模型的 Token 消耗分布
- **Weekly Trend** — 最近 7 天每日 Token 用量趋势折线图
- **Daily Intensity** — 6 个月每日 Token 用量强度日历热力图

## 范围切换

使用 `tab` 键切换统计范围：

- All Bots — 汇总所有 Bot 的用量
- 单个 Bot — 指定 Bot 的用量
- All Groups — 汇总指定 Bot 下所有群的用量
- 单个群 — 指定 Bot 下某个群的用量
