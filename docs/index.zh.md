# erii —— 有情感的 AI 群聊机器人

erii 是一个能参与群聊的 AI 机器人。它不只是简单地回复消息，而是像一个真实的群成员一样，有自己的情绪、记忆和性格。

## 快速安装

```bash
npm install -g erii-bot
```

<div class="grid cards" markdown>

-   :material-emoticon-outline:{ .lg .middle } __有情绪，会看气氛__

    ---

    erii 会根据聊天内容产生情绪变化。群聊热闹时它活跃，话题严肃时它安静，不会在不合适的场合乱说话。

-   :material-brain:{ .lg .middle } __会记住，越聊越懂你__

    ---

    erii 会记住群友说过的话和喜好。聊得越久，它越了解这个群，回复也越贴合群的风格。

-   :material-lightning-bolt:{ .lg .middle } __会主动，不只是被动应答__

    ---

    群聊冷场时，erii 会主动找话题暖场。遇到它感兴趣的内容，它会自己加入讨论——就像一个真实的群友。

-   :material-file-document-multiple:{ .lg .middle } __有规矩，不乱来__

    ---

    你可以用简单的 Markdown 文件给 erii 定规矩。不同群可以有不同的行为准则，erii 会严格遵守。

-   :material-puzzle:{ .lg .middle } __插件扩展__

    ---

    基于 PF4J 框架，通过安装插件扩展功能：语音合成、图片生成、音乐搜索、虚拟宠物等。

-   :material-power-plug:{ .lg .middle } __MCP 工具接入__

    ---

    通过 Model Context Protocol 接入外部工具服务，扩展 AI Agent 的能力边界。
