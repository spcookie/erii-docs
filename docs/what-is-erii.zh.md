# 什么是 erii

erii 是一个能参与 QQ 群聊的 AI 机器人。它不只是简单地回复消息，而是像一个真实的群成员一样，有自己的情绪、记忆和性格。

## 它能做什么

**有情绪，会看气氛**

erii 会根据聊天内容产生情绪变化。群聊热闹时它活跃，话题严肃时它安静，不会在不合适的场合乱说话。

**会记住，越聊越懂你**

erii 会记住群友说过的话和喜好。聊得越久，它越了解这个群，回复也越贴合群的风格。

**会主动，不只是被动应答**

群聊冷场时，erii 会主动找话题暖场。遇到它感兴趣的内容，它会自己加入讨论——就像一个真实的群友。

**有规矩，不乱来**

你可以用简单的 Markdown 文件给 erii 定规矩。不同群可以有不同的行为准则，erii 会严格遵守。

**能提醒，能总结**

设定时间后，erii 会定时提醒你。它还能自动总结你没空看的聊天记录，让你不错过精彩内容。

## 技术特色

| 功能       | 说明                                                     |
|:---------|:-------------------------------------------------------|
| 多 LLM 支持 | 支持 Google Gemini、OpenAI、Anthropic、OpenRouter 等多种 AI 模型 |
| MCP 工具扩展 | 通过 Model Context Protocol 接入外部工具服务，扩展 AI Agent 的能力      |
| 插件扩展     | 可以通过安装插件来增加新功能，如语音合成、图片生成、音乐搜索等                        |
| 多 Bot 管理 | 一个账号可以配置多个不同性格的 Bot，分配到不同的群                            |
| CLI 管理工具 | 提供命令行工具，方便配置和数据管理                                      |

## 怎么工作

erii 通过 NapCat 接入 QQ，收到消息后由 AI 模型决策如何回复。它的行为由规则文件约束，数据存储在本地，无需云端服务。

你可以通过插件给它增加各种能力——从发送表情包到生成图片，从语音合成到养成游戏。
