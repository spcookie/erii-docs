# erii-docs

erii 文档站点，基于 [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) 构建。

## 本地预览

```bash
pip install mkdocs-material mkdocs-static-i18n mkdocs-git-revision-date-localized-plugin mkdocs-git-committers-plugin-2 mkdocs-git-authors-plugin
mkdocs serve
```

访问 http://127.0.0.1:8088/

## 构建

```bash
mkdocs build
```
